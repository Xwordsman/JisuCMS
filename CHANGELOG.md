## [v3.0.3] - 2025-11-26 (LeCMS 最后版本)

### 新增功能
1. **Layuimini 升级**
   - 采用新版 Layui 2.8.15
   - 二次开发优化

2. **URL 函数优化**
   - 修改了分类、内容、标签的 URL 函数传递参数
   - URL 生成函数移动到 `urls_model.class.php`
   - 不含分类URL、内容URL、标签URL、评论URL

3. **搜索功能增强**
   - 增加了单独的搜索页（不是搜索结果页）
   - 模板中可使用 `{$search_url}` 链接

4. **自定义函数支持**
   - 新增用户自定义函数 `misc_func.php` 钩子
   - 可在插件中实现该钩子调用自定义函数
   - 参考 `jisucms/xiunophp/lib/misc.func.php` 中的说明

5. **单页评论功能**
   - 单页分类实现评论功能
   - 可直接用单页分类实现留言板功能
   - 参考 `page_message.htm` 模板

### Bug 修复
- 修复了多个小bug
- 优化了系统稳定性

### 插件兼容性
⚠️ **重要提示**: 涉及到 URL 函数的插件需要修改才能使用，包括：
- 分类筛选插件
- 推送插件
- 地图插件
- 其他使用 URL 函数的插件

## 版本说明

### 版本号规则
极速CMS 采用语义化版本号：`主版本号.次版本号.修订号`

- **主版本号**: 重大架构变更或不兼容的API修改
- **次版本号**: 向下兼容的功能性新增
- **修订号**: 向下兼容的问题修正

### 版本历史
- **v1.0.0** (2026-04-23): 极速CMS 品牌首发版本
- **v3.0.3** (2025-11-26): LeCMS 最后版本
- **v3.0.0 - v3.0.2**: LeCMS 3.0 系列版本

## 升级说明

### 从 LeCMS 3.0.3 升级到 JisuCMS 1.0.0

⚠️ **重要提示**: 这不是常规的版本升级，而是完整的品牌重塑。

#### 主要变更
1. **品牌名称**: LeCMS → JisuCMS
2. **核心目录**: lecms/ → jisucms/
3. **配置前缀**: le_ → jisu_
4. **版本号**: 从 3.0.3 重置为 1.0.0

#### 升级建议
- **新站点**: 直接使用 JisuCMS 1.0.0 全新安装
- **现有站点**: 可以继续使用 LeCMS，或进行完整迁移

#### 迁移步骤
1. 备份现有数据库和文件
2. 下载 JisuCMS 1.0.0
3. 全新安装 JisuCMS
4. 导入旧数据（需要调整表前缀）
5. 测试所有功能
