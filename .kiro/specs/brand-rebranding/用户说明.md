## ⚠️ 后台显示旧版本的问题

### 问题说明

您提到后台首页显示 "极速CMS 3.0.3 Release 20251126"，这是因为：

**版本信息存储在数据库中，不是代码文件中！**

后台首页的版本显示是从数据库的 `kv` 表中读取的，而您在品牌重塑之前已经安装过系统，所以数据库中还是旧版本信息。

### 解决方案

您有两个选择：

#### 方案1: 重新安装系统（推荐）✨

这是最简单、最彻底的方法：

1. **备份数据**（如果有重要内容）
2. **删除旧数据库** 或 **创建新数据库**
3. **访问安装程序**: `http://你的域名/install/`
4. **按照向导完成安装**

新安装会自动使用新版本号：
- 版本: 1.0.0
- 发布日期: 20260423
- 数据库名: jisucms
- 表前缀: jisu_

#### 方案2: 手动更新数据库 🔧

如果您不想重新安装，可以执行以下SQL语句：

```sql
-- 请根据您的实际表前缀修改（jisu_ 或 le_）
UPDATE jisu_kv 
SET v = REPLACE(
    REPLACE(v, '"version":"3.0.3"', '"version":"1.0.0"'),
    '"release":"20251126"', '"release":"20260423"'
)
WHERE k = 'cfg';
```

**如何执行SQL**:
1. 登录 phpMyAdmin 或其他数据库管理工具
2. 选择您的数据库
3. 点击 "SQL" 标签
4. 粘贴上述SQL语句（记得修改表前缀）
5. 点击 "执行"
6. 刷新后台页面

执行后，后台将显示：**极速CMS 1.0.0 Release 20260423** ✅
